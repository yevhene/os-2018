# Робота з семафорами

## Основні положення

### Мета
Ознайомитись з семафорами та маніпуляціями над ними.

## Матеріали

### Елемент семафора
```c
struct sembuf {
  ushort sem_num;   // номер елемента семафора
  short sem_op;     // значення конкретної дії над семафором
  short sem_flg;   //  прапорці
}
```
